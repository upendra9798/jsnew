xhr.readyState represents the state of the request. Its values are: 0 (UNSENT): The request is created but not yet opened. 1 (OPENED): The request has been opened with .open() but not yet sent. 2 (HEADERS_RECEIVED): The request has been sent, and the headers and status are available. 3 (LOADING): The response is being downloaded (partial data may be available). 4 (DONE): The entire response has been received and the request is complete.
